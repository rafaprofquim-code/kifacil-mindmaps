The pattern emerged!

You followed Mendeleev's reasoning: first the masses were arranged in increasing order; then, similar properties revealed columns. This organized repetition is periodicity. Today the modern table is organized by atomic number: for each number, there is a corresponding element — 1 hydrogen, 2 helium, 3 lithium, and so on.
